## 🛠️ Core Extensions & Official Ecosystem (by pvlib)
The core pvlib organization develops secondary tools built directly around or on top of the main pvlib-python engine:

* [pvanalytics](https://github.com/pvlib/pvanalytics): Focuses on quality control, data filtering, and feature labeling for PV system data.
* twoaxistracking: A specific package designed to simulate self-shading and tracking for two-axis solar collectors.
* [solarfactors](https://github.com/pvlib/solarfactors): A community implementation of the pvfactors bifacial irradiance model. [3] 

## 🌦️ Solar Power Forecasting & Weather Integration

* [PV-Prognose](https://github.com/tinoetzold/PV-Prognose): Calculates expected PV plant power output by connecting pvlib to German Weather Service (DWD) forecast data.
* solar-explorer: An interactive web application built with Marimo to design and simulate the yield of solar plants. [4, 5] 

## 💻 Open-Source Community Guides

* pv-tutorials: The official training repository containing end-to-end notebooks for lifecycle decisions, single-axis bifacial tracking, and system yields. [6] 

## 🔍 How to find more on GitHub
To see an updated, live list of all public code repositories depending on this package:

   2. For an exhaustive breakdown of thousands of open-source projects, click on the Dependency Graph under the repository's "Insights" tab to view every repository that lists pvlib in its requirements. [5]
